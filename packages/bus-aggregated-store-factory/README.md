# Comunica Bus Aggregated Store Factory

[![npm version](https://badge.fury.io/js/%40comunica%2Fbus-aggregated-store-factory.svg)](https://www.npmjs.com/package/@comunica/bus-aggregated-store-factory)

A comunica bus for aggregated-store-factory events.

This module is part of the [Comunica framework](https://github.com/comunica/comunica),
and should only be used by [developers that want to build their own query engine](https://comunica.dev/docs/modify/).

[Click here if you just want to query with Comunica](https://comunica.dev/docs/query/).

## Install

```bash
$ yarn add @comunica/bus-aggregated-store-factory
```
