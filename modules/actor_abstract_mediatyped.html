Module actor-abstract-mediatyped

Comunica Abstract MediaTyped Actor

A actor for handling actions based on mediatypes, such as content negotiation

This module is part of the Comunica framework, and should only be used by developers that want to build their own query engine.

Click here if you just want to query with Comunica.

Install

 
$ yarn add @comunica/actor-abstract-mediatyped

Usage

This package exposes ActorAbstractMediaTyped and ActorAbstractMediaTypedFixed that provide utility methods for actors that handle media types.
